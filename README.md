# System Setup & User Guide

## System Requirements

### Operating System
- Windows 10 or higher (64-bit recommended)

### Software
- Visual Studio Code  
- Python (v3.x)  
- Web Browser (Google Chrome, Brave, etc.)

### Internet Connection
- Required for initial setup and dependency installation  

---

## ⚙️ Installation Guide

### 1. Install Visual Studio Code
1. Download and install Visual Studio Code  
2. Install extensions (if needed):
   - Live Server (for static version)

---

### 2. Install Python
1. Download and install Python **3.10 or higher**  
2. ✔️ Make sure to check **"Add Python to PATH"**

---
